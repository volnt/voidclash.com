1. Configuration File customization and highlights

Heads up! Here are defined all elements needed by the server to run properly. Anyway, you can change the order of the element and add more configuration in the file !

Configuration File structure

The configuration file is designed following the json data structuration. Its name is config.json

Example:
{
    "config": "Basic",
      "game": {},
      "horde": {},
      "player": {},
      "server": {},
      "weapons": {}
}

2. Game configuration

Configuration File structure

The configuration file is designed following the json data structuration. Its name is config.json

Example:
"game": {
    "friendly_fire_own": 0,
    "friendly_fire_team": 0,
    "round_nb": 15,
    "speed": 20,
    "zoom": 1.6,
    "map": "square.json",
    "map_duration": 40,
    "mode": "CTF",
    "warmup_duration": 10
}

Attributes overview

  int         speed;
  float       zoom;
  int         friendly_fire_own;
  int         friendly_fire_team;
  int         round_nb;
  std::string map;
  int         map_duration;
  std::string mode;
  int         warmup_duration;

Game attributes

Throught this table, you can find all the weapon parameters that you can use in your configuration file.


boolean: true or false

int: (integer) from -2 147 483 648 to
2 147 483 647

float: (real) from -3.4*10^-38 to 3.4*1038

string: sequence of characters

Attr Name Type Description
speed int General speed of the game
zoom float Maximum value of the zoom
friendly_fire_own int Own friendly fire, get damage from your weapons, from (0-100 for 1.x usage, upgrade for more)
friendly_fire_team int Team friendly fire, give and take damage from your teamates weapons, from (0-100 for 1.x usage, upgrade for more)
round_nb int Number of rounds
map string Default map loaded, useful to change the map from the web interface
map_duration int Map duration in seconde
mode string >Default mode loaded, useful to change the map from the web interface
warmup_duration int Warmup duration in seconde

6. Weapons

General Weapon structure

An example of a weapon in the config.json configuration file.

Basic weapon:
  "weapons": {
       "G_Example": {
            "category": "GRAVITY",
            "speed": 0,
            "size": 500,
            "bounce": 1,
            "duration": 2,
            "fire_rate": 0.8,
            "energy_cost": 200,
            "pushback_fire": 0,
            "pushback_other": 0,
            "damage": 200,
            "slow": 80,
            "collide_walls":  false
        }
    }
Composed weapon:
  "weapons": {
       "G_Example": {
            "category": "GRAVITY",
            "speed": 0,
            "size": 500,
            "bounce": 1,
            "duration": 2,
            "fire_rate": 0.8,
            "energy_cost": 200,
            "pushback_fire": 0,
            "pushback_other": 0,
            "damage": 200,
            "slow": 80,
            "collide_walls":  false
        }

        "T_Shotgun": {
            "capacity": 1,
            "category": "TURRET",
            "detection_range": 1000,
            "duration": 5,
            "energy_cost": 700,
            "fire_rate": 1,
            "life": 1000,
            "size": 30,
            "speed": 0,
            "sub_weapon": "G_Example"
        }
      }

Attributes overview

  std::string   name;         // Name of the weapon
  std::string   category;     // Category of the weapon (ie. Bullet / Rocket...)
  int     energy_cost;        // All - Energy cost of the weapon
  int     init_energy_cost;   // Shield - Value of the drain energy when activating shield on the first frame
  int     damage;             // Bullet / Bomb / Rocket / Gravity - Damage
  int     speed;              // All - Velocity
  float   duration;           // All - Duration
  float   fire_rate;          // All - Firerate of the weapon
  int     size;               // All - Size of the projectile
  int     size_explosion;     // Rocket / Bomb - Size of the explosion
  int     pushback_fire;      // All - Pushback power on player when he fires
  int     pushback_other;     // All - Pushback power on other player when they get hit
  int     acceleration;       // Bomb / Shield - Give a speed boost
  int     chain;              // Bomb / Rocket - Is triggered when in range of another explosion
  int     drain_energy;       // All - Value of the drain energy when an enemy is hit
  int     shot_nb;            // Bullet - Nb of shots (ie. Shotgun)
  int     angle;              // Bullet - Spread angle of shots (ie. Shotgun)
  bool    collide_walls;      // All - true to enable collision with walls
  int     bounce;             // All - bounce agaisn't walls
  int     detection_range;    // Turret - Range of detection
  int     life;               // Turret - Turret life // 0 = immortal
  int     capacity;           // Turret - Nb of elements
  int     slow;               // Bullet / Bomb / Rocket / Gravity - Power of slow in percent (0 - 100)
  int     slow_duration;      // Bullet / Bomb / Rocket - Duration of slow

  std::string   subWeaponName;    // Just used for parsing
